# HorizonTechX Network Sniffer

Python-based network packet sniffer using Scapy for real-time packet capture, protocol analysis, and network monitoring.

## Overview

This project is a Python-based network packet sniffer developed using the Scapy library. It captures live network traffic, analyzes packets in real time, and displays important information such as IP addresses, protocols, ports, timestamps, and packet statistics.

## Features

* Capture live network traffic
* Detect TCP, UDP, and ICMP protocols
* Display Source and Destination IP addresses
* Display Source and Destination Port numbers
* Display packet timestamps
* Display packet lengths
* Generate protocol statistics
* Real-time packet monitoring

## Technologies Used

* Python 3
* Scapy
* Datetime Module
